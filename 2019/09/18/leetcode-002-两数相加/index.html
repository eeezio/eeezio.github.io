leetcode 002 两数相加

Posted on 2019-09-18
大佬的解法很是优美

题目描述：

给出两个 非空 的链表用来表示两个非负的整数。其中，它们各自的位数是按照 逆序 的方式存储的，并且它们的每个节点只能存储 一位 数字。

如果，我们将这两个数相加起来，则会返回一个新的链表来表示它们的和。

您可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

解题思路：

这题首如果遍历两个链表求出链表表达的数的和，然后再通过这个和构建新的链表是不行的，因为在初始给定的两个链表很长的情况下，即使使用long long类型还是不能表达足够大的数。所以我们应当对两个链表按位进行计算。计算时要注意以下几个要点。

  1. 两个链表的结点相加时，一定要注意要用一个变量保存进位。
  2. 如果两个链表不等长，可以将进位信息在较长的链表上更新完毕以后，将剩余的长链表直接链在我们需要返回的链表上。
  3. 如果需要返回的链表的最后一个节点有进位的话，要记得创造一个新节点，链在链表上。
