2.

分析：

  1. 从 orq %r8,%rax 可以推理出，result存在rax寄存器中。
  2. 从 test %rdx,%rdx 以及 jne .L3 可以推理出，mask存在rdx寄存器中，从 movl $1,%edx 可以推理出，mask的初始值为1，以及判断循环是否应该中止的条件是 mask！=0。
  3. 从 saql %cl,%rdx 可看出，每轮循环mask都要左移，并且左移的位数存在cl寄存器中，cl只占1个字节，可以通过ecx寄存器&0xFF获得，而ecx的值来自esi寄存器，也就是说ecx的初始值是n，所以每轮循环的左移操作是 mask << (n & 0xFF)
  4. 从 movq %rdi,%r8    andq %rdx,%r8    orq %r8,%rax三句汇编代码可以很容易推理出来循环内对result做的操作是 result |= x & mask

4.

A.

$CNT=7$
分析：
结合int n=bp->first+bp->last;以及其对应的汇编mov 0x120(%rsi),%ecx    add(%rsi),ecx可以推理出，a[CNT]一共占了0x120-0x8=280个字节(虽然first占4个字节，但是出于边界对齐的考虑，编译器分配了8个字节),这几句汇编lea (%rdi,rdi,4),%rax    lea (%rsi,%rax,8),%rax    mov 0x8(%rax),%rdx其实做的就是去寻找a[i],它的地址位于bp+8+40i，加8的原因同理。从+40i我们就可以看出来了，一个a_struct占了40个字节。这样就很容易计算CNT的值等于$280\div40=7$

B.

分析:
从mov %rcx,0x10(%rax,%rdx,8) 可以看出，ap->x[ap->idx] 的地址位于bp+40i+16+8idx,可以写为bp+40i+8+8+8idx,其中加的两个8一个是int first(出于边界对齐的考虑分配了8个字节)，另一个是idx，为long型变量占8个字节。从8idx可以看出，数组x的每个元素占8个字节，为long型数组，从而推理得出a_struct的结构。
